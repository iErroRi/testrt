# testrt
## Installing

`sudo mkdir /srv/testrt`  
`sudo chmod 777 /srv/testrt` для простоты    
`cd /srv/testrt`  
`git clone git@github.com:iErroRi/testrt.git .`  
`docker-compose build`  
`docker-compose up`
